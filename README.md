# Ejecutar en desarrollo
1. Clonar el repo
2. Ejecutar
```
yarn install
```
3. Tenere nest cli instalado
```
npm i -g @nestjs/cli
```
4. Levantar la DB

```
docker-compose up -d
```
## Stack usado
* MongoDB
* Nestjs
